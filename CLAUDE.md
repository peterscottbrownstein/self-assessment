# CLAUDE.md

This repository uses:

- `README.md` for the human-facing project overview, setup, and day-to-day usage
- `AGENTS.md` as the primary assistant handoff and workflow guide

If you are working in this repo as an assistant:

1. Read `README.md` for product and repository context.
2. Read `AGENTS.md` for assistant-specific guidance, architecture notes, and handoff rules.

`CLAUDE.md` is intentionally kept minimal so the shared assistant guidance lives in one place instead of drifting across multiple files.
